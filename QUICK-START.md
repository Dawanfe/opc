# 微信公众号配置 - 快速指南

## 📋 第一步：获取配置（10分钟）

### 1️⃣ 配置网页授权域名

登录 https://mp.weixin.qq.com

**路径**：设置与开发 → 公众号设置 → 功能设置 → 网页授权域名

**操作**：
- 点击"设置"
- 输入：`weopc.com.cn`
- 下载验证文件：`MP_verify_xxxxxxxxxx.txt`
- 记录文件名和打开查看内容

---

### 2️⃣ 获取AppID和AppSecret

**路径**：设置与开发 → 开发 → 基本配置

**操作**：
- 复制 **开发者ID(AppID)**
- 点击 **开发者密码(AppSecret)** 的"重置"
- 扫码确认
- ⚠️ **立即复制保存**（只显示一次）

---

### 3️⃣ 配置IP白名单

同一页面，IP白名单 → 修改

**添加**：`101.200.231.179`

---

## 🚀 第二步：一键配置（1分钟）

在项目目录运行：

```bash
./setup-wechat-mp.sh
```

按提示输入：
- 公众号AppID
- 公众号AppSecret
- 验证文件名
- 验证文件内容

脚本会自动：
1. ✅ 创建验证文件
2. ✅ 更新环境变量
3. ✅ 部署到生产环境

---

## ✅ 第三步：完成验证（30秒）

部署成功后：

1. 访问 `https://weopc.com.cn/MP_verify_xxx.txt` 确认文件可访问
2. 回到微信公众平台，点击【确定】完成域名验证

---

## 🎉 测试效果

**移动端**：
1. 手机访问 https://weopc.com.cn
2. 点击"微信授权登录"
3. 跳转授权页面 → 允许 → 登录成功 ✅

**PC端**：
1. 访问 https://weopc.com.cn
2. 点击"微信授权登录"
3. 跳转扫码页面 → 扫码 → 登录成功 ✅

---
