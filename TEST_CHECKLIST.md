## 🐛 故障排查

### 问题1: 服务器未启动
**症状**: `❌ 开发服务器启动失败`
**解决**:
```bash
cd next
npm run dev
# 等待启动后，新终端运行测试
npm test
```

### 问题2: 数据库锁定
**症状**: `database is locked`
**解决**:
```bash
pkill -f next
rm -f next/data/opc.db-shm next/data/opc.db-wal
npm run dev
```

### 问题3: 端口被占用
**症状**: `port 3000 already in use`
**解决**:
```bash
lsof -ti:3000 | xargs kill -9
npm run dev
```

### 问题4: 测试数据冲突
**症状**: `该手机号已注册`
**解决**:
```bash
npm run test:seed clean
npm test
```
